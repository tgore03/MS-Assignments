CS 572 - Lab 1

How to complile the code.
1. Open the control panel in the current directory.
2. execute javac WebSearch.java
3. execute java Websearch intranet# <search method>   //# represent 1, 5 or 7
4. Search methods:
	a. breadth - breadth first search
	b. depth - depth first search
	c. best	- best first search
	d. beam - beam search
5. For beam search, the beam width will be asked at the run time.
